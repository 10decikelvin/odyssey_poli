 This project is designed to control an autonomous robot, enabling it to navigate towards a target position while avoiding obstacles. It utilizes PID control for precise movement and implements various algorithms for path planning and obstacle avoidance.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{PID Control}}\+: Utilizes Proportional-\/\+Integral-\/\+Derivative (PID) control to manage the robot\textquotesingle{}s speed and direction accurately.
\item {\bfseries{Serial Communication}}\+: Communicates with external devices or computers through serial communication, allowing for dynamic target position updates.
\item {\bfseries{Encoder Feedback}}\+: Uses encoders for feedback on the robot\textquotesingle{}s movement, ensuring accurate positioning and speed control.
\item {\bfseries{Low-\/\+Pass Filtering}}\+: Implements low-\/pass filtering to smooth out the control signals, reducing noise and fluctuations.
\item {\bfseries{Circle Tracking}}\+: Includes an algorithm for circle tracking, enabling the robot to follow curved paths smoothly.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Components}{Components}}\label{md__r_e_a_d_m_e_autotoc_md2}
